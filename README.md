🛒 Supermarket Sales Intelligence Dashboard
Turn Raw Retail Data into Decisions, Not Just Charts
🚀 What This Project Does

This is not just a dashboard — it’s a decision-support system for retailers.

It takes messy, real-world sales data and automatically converts it into:

Clear business insights
Customer behavior analysis
Future sales predictions

All through a simple, interactive web interface.

🎯 Why I Built This

Most small retailers already have data — but they don’t have the tools or expertise to use it.

I built this to solve a practical gap:

“How can someone upload raw data and instantly get insights they can act on?”

The focus was not just on machine learning, but on usability, automation, and real-world impact.

⚡ Key Highlights


🧠 Intelligent Data Handling

Automatically detects key columns (date, sales, category)
Cleans messy datasets (missing values, duplicates, invalid entries)
Even derives total sales (quantity × price) if not provided

📊 Automated Business Insights

Category-wise revenue analysis
Top-performing products
Correlation heatmaps (to uncover hidden patterns)
Profit analysis (if available)

🎯 Interactive Decision Controls

Filter data by:
Date range
Product category
City/location
Instantly see how decisions affect outcomes

👥 Customer Behavior Analysis

Segment users by:
Gender
City
Payment method
Age
View:
Total sales
Transactions
Purchase patterns

📈 Built-in Sales Forecasting

Predict future demand using:
Prophet (primary model)
ARIMA (fallback)
Visualize trends and identify peak sales periods
Helps with:
Inventory planning
Pricing strategy

📦 Export-Ready Outputs

Download cleaned datasets
Export forecast results
Use insights outside the platform

🎨 Product-Focused UI

Clean, modern dashboard (Streamlit)
Light/Dark theme toggle
Designed for non-technical users

🛠️ Tech Stack

Layer	Tools Used
Frontend	Streamlit
Data Processing	Pandas, NumPy
Visualization	Plotly
Forecasting	Prophet, ARIMA (statsmodels)
Language	Python

🧩 How It Works

Upload your dataset (CSV/Excel)
System automatically cleans and structures the data
Generates insights + visualizations
Applies forecasting on time-series data
Displays results and allows export

💡 What Makes This Different

Not just ML → end-to-end product
Handles real-world messy data, not clean datasets
Built for usability, not just accuracy
Focus on decision-making, not just visualization

🔮 Future Improvements
